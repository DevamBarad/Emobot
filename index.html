Multimodal control meets genuine artificial empathy. Powered by LLaMA 3 & Quantrillion OS.

Hybrid Intelligence


The uniqueness of this project lies in combining multimodal control (gestures, mobile commands, and voice) with emotional intelligence in a single system. Unlike conventional robots, it not only executes commands but also detects and expresses emotions using a locally running LLaMA 3 model.


"Not just a machine, but a companion."

THE TECH STACK

LLaMA 3.1 1B Python NLP OpenCV (Face/Gesture) Arduino Mega OLED & Servos Quantrillion OS

Beyond Practicality


This hybrid design makes the robot more human-like, empathetic, and practical for real-world applications in healthcare, education, and assistive technologies.


Offline Privacy: Real-time response with no data leaving the device.
